# Odin Recipes

A simple HTML and CSS project for practicing basic web development. This site displays a few delicious recipes using structured semantic HTML and custom styling.

## Features

  • Home page with welcome message
  • Navigation bar with recipe links
  • Individual pages for each recipe
  • Clean and responsive layout using CSS Flexbox

## Technologies Used

  • HTML5
  • CSS3
  • Visual Studio Code
  • Git & GitHub

## How to Use

  1. Clone the repository: git clone :
    https://github.com/seu-usuario/odin-recipes.git
    https://github.com/seu-usuario/odin-recipes.git

  2. Open `index.html` in your browser to view the site.
